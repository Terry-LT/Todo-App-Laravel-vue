<h2>Frameworks which are used in this Project:</h2>

<h3>Backend Side:</h3>
<ul>
    <li>Larave 8(php)</li>
</ul>

<h3>Frontend  Side:</h3>
<ul>
    <li>Vue 2 (javascript)</li>
    <li>Bootstrap 5 (css)</li>
</ul>

<h3>Features:</h3>
<ul>
    <li>Laravel api with vue js</li>
    <li>Vue Router</li>
</ul>

<h4>To get start</h4>
<ol>
    <li>php artisan migrate</li>
    <li>npm install</li>
    <li>php artisan serve</li>
    <li>npm run watch</li>
</ol>
